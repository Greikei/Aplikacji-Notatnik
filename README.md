# Aplikacji-Notatnik      Note Helper


## Funkcjonalność

**Tworzenie notatek**
**Dodawanie zdjęć do notatek**
**Geolokalizacja**
**Tryb Offline**
**Hosting HTTPS dzieki github pages**

## Użyte Technologie

**HTML5**
**JavaScript**
**Api Przeglądarkowe**
    * GeoLocation
    * MediaRecorder
    * FileReader
    * LocalStorage
**Service Worker**
**Zewnętrzne API**: (https://www.bigdatacloud.com/)



## Instrukcja uruchomienia

### Metoda lokalna: localhost (Tylko http)
1. Pobierz repozytorium.
2. Zainstaluj rozszerzenie: Live Server w Visual Studio Code.
3. Klikając prawym klawiszem myszki na plik index.html wciskamy przycisk "Open with Live Server", lub alternatywnie wciskamy skrót klawiszowy (Alt + L Alt + O).

### Metoda HTTPS
1. Otwieramy stronę: (https://greikei.github.io/Aplikacji-Notatnik/).
2. W polu adresu URL, po prawej stronie widzimy ikonę prostokąta z małą strzałka w doł "Install Note Helper".
3. Klikamy "install" potwierdzając instalacje.
4. Otwieramy aplikacje znajdującą sie na pulpicie komputera.
